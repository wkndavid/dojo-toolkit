_data-dojo-attach-point_ :

==Um ponto de anexação informa ao renderizador de modelo que quando um elemento DOM é criado com um atributo data-dojo-attach-point definido, para definir o valor desse atributo como uma propriedade do seu widget para ser uma referência ao elemento DOM criado.==

_dojo-data-attach-event_ :

==Além de anexar pontos, o sistema de modelos Dijit oferece uma maneira de anexar eventos DOM nativos a métodos em seu widget personalizado. Isso é feito através do uso do atributo de dados HTML5 `data-dojo-attach-event`.
